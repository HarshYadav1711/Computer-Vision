# Input videos

Put your side-on or semi side-on cricket batting or bowling video here (e.g. `batting.mp4`).

Then run from the project root:

```bash
python run_pipeline.py --video videos/batting.mp4
```

Or use just the filename if the file is in this folder:

```bash
python run_pipeline.py --video batting.mp4
```

If the filename has spaces, quote it: `python run_pipeline.py --video "videos/Volleyball spike.mp4"`
